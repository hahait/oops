## oops
    运维管理平台

### 功能说明
    - 其他功能继承 WS-OPS
    - 以应用为中心的监控
    - 告警统一管理

### 运行环境   
    * Python 3.6.3
    * Django 1.11.7
    * djangorestframework 3.8.2 
    * vue 2.5.17

### 启动方式
    ```
    略
    ```
### 功能详细说明

    - 以应用为中心的监控
        1. 根据选择的应用,来查看该应用下所有主机的系统级监控(如 CPU、内存、磁盘、网络等)
        2. 监控数据来源于 zabbix
        3. 使用 echarts 绘图

    - 告警统计管理
        1. 收集 zabbix 和 应用监控系统 cat 及 rocketmq 等告警信息，做统一的处理: 如告警跟踪，告警升级，告警分组发送，告警统计
